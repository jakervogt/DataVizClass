Summary Latitude vs. X

This is the latitude analysis dashboard! This site analyzes many different comparisons of over 500 cities. These comparisons include Cloudiness levels, Windspeeds, Humidity levels, and max temperatures for all these different cities. The tool used to create these plots was Matplotlib in jupyter notebook. The graphs shown on this website show the different variables above compared to the specific latitude of each city.
